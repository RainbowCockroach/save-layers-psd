# Save layers PSD

This app save layers of psd file to separate png images while maintaining the folder structure.

To use:

- Download (in `Releases` section, download for Windows or Mac). Additional step on Mac: unzip, click on the file and allow running in `Privacy & Security` settings
- Drag the psd file in the app icon
- Two export mode:
  - Paper size: All result images have the same dimension as the whole canvas
  - Actual size: Result images have the size that fits their content
